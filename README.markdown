# Jupitercandle

A jump'n'run in the distant future. 

## Usage

* run (left/right arrow)
* jump (up arrow)
* dash (c)
* restart with new level (r)

## Installation

```
git clone
(ql:quickload :jupiter-candle)
(jupiter-candle:main)
```
